1. 访问 OpenAI 官网：[OpenAI 注册页面](https://platform.openai.com/signup)，点击右上角的“Sign up”按钮。

3. 验证邮箱地址：点击发送到邮箱的链接，激活账号。
4. 登录账号后，创建 API KEY。点击左侧栏的“API Keys”，然后点击“Create new secret key”，创建后立即复制并保存（关闭弹窗后将无法查看）。
